1.Поиск 
    - линейный поиск — O(n) 
    - бинарный поиск — O(log n) 
2.Сортировка    
    - пузырьковая — O(n²)   
    - сортировка выбором — O(n²)    
    - сортировка вставками — O(n²)  
    - быстрая сортировка — в среднем O(n log n) 
    - сортировка слиянием — O(n log n)  
    - пирамидальная сортировка — O(n log n) 
3.Обход коллекций   
    - Array Traversal;  
    - Linked List Traversal;    
    - Tree Traversal;   
    - Graph Traversal.  
4.Рекурсивные алгоритмы 
    - факториал 
    - числа Фибоначчи   
    - обход дерева  
5.Разделяй и властвуй   
    - бинарный поиск    
    - Quick Sort    
    - Merge Sort    
6.Жадные алгоритмы(на каждом шаге выбирается локально лучший вариант)   
    - Dijkstra’s Algorithm; 
    - Prim’s Algorithm; 
    - Kruskal’s Algorithm;  
    - Huffman Coding;   
    - Activity Selection Algorithm. 
7.Динамическое программирование(признаки DP: задача разбивается на повторяющиеся подзадачи; промежуточные результаты сохраняются)   
    - Fibonacci with Dynamic Programming;   
    - 0/1 Knapsack; 
    - Longest Common Subsequence;   
    - Coin Change;  
    - Edit Distance.    
8.Перебор и возврат — backtracking(описание: перебираются варианты, неподходящая ветка отменяется. Пример: поиск выхода из лабиринта)   
    - Maze Solving; 
    - Sudoku Solver;    
    - N-Queens; 
    - Graph Coloring;   
    - Permutation Generation.   
9.Алгоритмы на графах   
    - обход в ширину — BFS  
    - обход в глубину — DFS 
    - алгоритм Дейкстры 
    - топологическая сортировка 
10.Алгоритмы на деревьях    
    - прямой, симметричный и обратный обход 
        - Pre-order Traversal;  
        - In-order Traversal;   
        - Post-order Traversal. 
    - поиск в бинарном дереве   
        - Level-order Traversal;    
        - Tree Insertion;   
        - Tree Deletion;    
        - Tree Balancing.   
11.Хеширование(вычисление хеша, поиск в Dictionary и HashSet в среднем за O(1)) 
    - Hash Function;    
    - Separate Chaining;    
    - Open Addressing;  
    - Linear Probing;   
    - Quadratic Probing;    
    - Double Hashing.   
12.Строковые алгоритмы(Задача: поиск подстроки. Операция: сравнение строк, Отдельная задача с простым алгоритмом: проверка палиндрома)  
    - Naive String Search;  
    - Knuth–Morris–Pratt;   
    - Rabin–Karp;   
    - Boyer–Moore.

    3. Обход массивов и связанных списков.  
    4. Простые сортировки.  
    5. Рекурсия.    
    6. Быстрая сортировка и сортировка слиянием.    
    7. Стек, очередь, деревья.  
    8. DFS и BFS.   
    9. Графовые алгоритмы.  
    10. Хеширование.    
    11. Жадные алгоритмы.   
    12. Backtracking.   
    13. Динамическое программирование.
